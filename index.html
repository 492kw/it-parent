Basic setup parents can start with

A strong home setup does not need to start with expensive subscriptions

The practical foundation is usually the home router plus the free controls already available on Android and iPhone. The router controls the household network. The phone-level tools control screen time, apps, and content on the child’s device.

This gives families a realistic starting point: use the router to manage devices on Wi-Fi, then use Google Family Link or Apple Screen Time to manage the child’s phone directly.

Home router / network control

Ubiquiti UniFi Express 7 (UX7)

The UX7 is a compact UniFi cloud gateway with integrated Wi-Fi 7. It includes routing, Wi-Fi, firewall, traffic identification, content filtering, ad blocking, intrusion prevention, VLAN/subnet traffic segmentation, guest isolation, and internet quality reporting.

  • Useful for seeing what devices are on the home network.
  • Helps create firewall and filtering rules for devices or groups of devices.
  • Can separate household devices into different networks, such as parent devices, child devices, and guest devices.
  • Provides stronger visibility and control than a basic ISP modem/router, while also replacing or improving household Wi-Fi.
Android / Google accounts

Google Family Link

Family Link is Google’s free parental control app. It helps parents set screen-time limits, downtime, school-time schedules, app limits, app approvals, location sharing, and Google account settings for children and teens.

  • Best starting point for Android phones and supervised Google accounts.
  • Can set daily device limits and individual app limits.
  • Can block or approve apps and manage Google service settings.
  • Can help locate a child’s device when location sharing is enabled.
iPhone / iPad

Apple Screen Time

Screen Time is built into iOS and iPadOS. With Family Sharing, a parent or guardian can manage a child’s screen time from their own Apple device, including downtime, app limits, content restrictions, communication limits, and Screen Time passcodes.

  • Best starting point for iPhones and iPads.
  • Can schedule downtime and set app/category limits.
  • Can block age-inappropriate content, apps, downloads, and websites.
  • Can show reports on apps and websites used most.

Suggested free configuration path

Network-level controls

  1. List every device in the house: parent phones, child phones, tablets, laptops, consoles, TVs, and guest devices.
  2. Rename each device in the router so the family can tell who owns what.
  3. Create a separate child device group or network where possible.
  4. Apply sensible router rules: block risky categories, restrict unnecessary device-to-device access, and use schedules for internet access where appropriate.
  5. Review traffic and usage patterns so the family can adjust rules based on real behaviour, not guesswork.

Phone-level controls

  1. For Android, install and configure Google Family Link with the child’s supervised Google account.
  2. For iPhone, set up Apple Family Sharing and Screen Time for the child’s Apple Account.
  3. Set daily limits, downtime, app limits, and content restrictions.
  4. Keep essential apps available, such as phone, maps, health, accessibility, school, or emergency communication apps.
  5. Review the setup weekly at first, then adjust as the child’s age, school needs, and behaviour change.
